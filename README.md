An update of the WebGL Teaching App build with :heart: @uzh

This app is using [@react-three/fiber](https://github.com/react-spring/@react-three/fiber) reconciler under the hood.

## Development

**install packages**

```bash
npm install
```

**development**

spins up a server on PORT 3000

```bash
npm run start
```

**build**

```bash
npm run build
```
